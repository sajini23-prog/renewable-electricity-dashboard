# Renewable Electricity Dashboard

This project is a Streamlit dashboard developed for the 5DATA004C Data Science Project Lifecycle coursework.

## Dataset
Renewable electricity output (% of total electricity output)  
Source: World Bank

## Features
- KPI summary cards
- Top and bottom country ranking
- Country comparison analysis
- Trend over time
- World map
- Filtered data table
- Insight summary
- Change between two years analysis

## Tools Used
- Python
- Streamlit
- Pandas
- Plotly
- GitHub
